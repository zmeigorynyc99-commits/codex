# DevOps + Network Engineering Course

A beginner-to-senior course. First delivery includes the first 10 complete lessons, labs, exercises, solutions, homework folders, and a simple local website.

## Run locally

```bash
docker compose up --build
```

Open http://localhost:8080.

## Structure

- `botera.md` course runbook
- `lessons/` complete lessons
- `labs/` guided lab support files
- `exercises/`, `solutions/`, `homework/` practice areas
- `site/` minimal Markdown-rendering website
